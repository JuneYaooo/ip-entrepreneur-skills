# ip-entrepreneur-skills

**Claude Code / Codex Skill。一句「帮我诊断这个 IP」，自动六维度深度诊断 —— 定位×人设×内容×数据×变现×长期，精华版 + 全面版双输出。**

## ✨ 能做什么

- 📊 **六维度深度诊断** — 定位 × 人设 × 内容 × 数据 × 变现 × 长期，覆盖 IP 创业全链路
- 🌐 **全网信息聚合** — 配合 opencli 联网搜索，自动收集该 IP 的多平台数据、媒体报道、社区口碑
- 🔍 **赛道竞品全览** — 自动对焦 5-8 个同赛道玩家，拆解人设、内容、变现、直播策略
- 📈 **数据驱动决策** — 收入公式拆解、转化漏斗诊断、增长阶段定位，用数据替代直觉
- 💰 **变现路径规划** — 定价策略、产品矩阵、营收模型，从品牌影响力到收入的完整转化路径
- 🗺️ **死亡区分析** — 定位你在 IP 生命周期中的位置，识别核心风险并给出穿越策略
- ⚡ **双模输出** — 精华版（5 分钟读完+行动清单）+ 全面版（完整六维度分析），同时生成

## 🚀 安装

### 方式一：让 AI 自己装（推荐）

把下面这段 prompt 丢给你的 AI 助手（Claude Code / Codex / Cursor 都行），它会自动 clone 仓库并完成安装：

```
帮我安装 ip-entrepreneur-skills：
https://github.com/JuneYaooo/ip-entrepreneur-skills
```

### 方式二：手动安装

```bash
git clone git@github.com:JuneYaooo/ip-entrepreneur-skills.git
# Claude Code 会在当前目录的 .claude/skills/ 下自动识别 skill
# 或者放到 ~/.claude/skills/ip-entrepreneur/
```

> 🧠 **强烈建议同时安装 [opencli](https://github.com/jackwener/opencli)**，为 Claude Code 开启联网搜索。有了网络访问能力后，AI 诊断时可以实时搜索该 IP 在所有平台的公开信息，而非仅凭你提供的描述进行分析。

## 🛠 怎么用

安装后，直接在 Claude Code 里说人话就行。推荐搭配 opencli 启用联网搜索，效果最佳。

### 模式 A — 诊断现有 IP

> 帮我诊断 `https://github.com/alchaincyf` 这个人的 IP，从定位、人设、内容、数据、变现、长期六个维度分析。先全网搜一下 ta 的公开信息。

丢一个 GitHub 主页、社交媒体链接，甚至一个名字都行。AI 会用 opencli 全网搜罗该 IP 的公开信息，自动生成报告。

### 模式 B — 从 0 到 1 起步

> 我想在 [赛道] 做个人 IP，帮我全网搜一下这个赛道的头部玩家，然后做定位分析、人设设计和前 30 条选题规划。

### 模式 C — 增长突破

> 我的 IP 目前 [现状]，数据卡在 [瓶颈]，帮我全网搜一下竞品怎么做的，然后诊断问题并给出优化方案。

**核心方法论**内置于 SKILL.md 中，包含六维度框架、金句话术、实战 checklist 等完整知识体系。`references/` 目录下另有 6 份深度参考资料，agent 在诊断时会按需查阅。

# 商业变现体系

> IP 不是自媒体，IP 是商业模式。做 IP 就是做公司，一开始就要考虑盈利。

## 一、IP 变现三类别

### 1. 卖知识
- 课程（录播/直播/训练营）
- 咨询（1v1/1vN）
- 社群（付费圈子）
- 出版（书/电子内容）

### 2. 卖商品
- 自营产品（有供应链）
- 带货/分销（无供应链，纯佣金）
- 联名定制（IP 授权+供应链合作）

### 3. 卖机会
- 加盟/代理
- 线下大会/活动
- 投资/合作
- 人才推荐

## 二、产品设计——上中下游商机框架

从供应链→到个人技能→到终端用户，三个方向梳理可变现方向：

### 上游（供应链/资源端）
- 你有货源渠道吗？
- 你有独家资源吗？
- 你有稀缺信息吗？

### 中游（个人经验/技能）
- 你有什么专业技能？
- 你走过什么独特的路径？
- 你有什么信息差？

### 下游（终端用户需求）
- 你的粉丝遇到了什么问题？
- 他们愿意为什么付费？
- 市场上有什么未被满足的需求？

### 案例
- 养花达人 → 卖花（商品）+养护课程（知识）+花园设计（服务）
- 北大博士做抖音 → 抖音创业课（知识）+1v1 咨询（服务）+工具推荐（分销）

## 三、产品开发五步流程

```
①初步调研（用户需求验证）
  ↓
②MVP 海报（一张图说清楚卖什么）
  ↓
③课程大纲（结构化的交付内容）
  ↓
④完善内容（录制/写稿/做课件）
  ↓
⑤配套服务（答疑/社群/作业批改）
```

### 关键原则
- **先跑通用户效果，再规模化推广**
- 录播课是在训练营验证了方法论之后才做
- 路径：训练营（验证 PMF）→ 录播课（规模化）→ 社群/咨询（高客单价）

## 四、销售六手段

| 手段 | 内容 | 适用场景 |
|------|------|---------|
| 卖情怀 | 你的初心、使命、梦想 | 品牌建设、铁粉凝聚 |
| 卖经历 | 你走过的路、踩过的坑 | 建立信任、降低决策门槛 |
| 卖认知 | 独到的见解、方法论 | 知识付费核心、高溢价 |
| 卖好评 | 用户案例、真实反馈 | 打消顾虑、促进下单 |
| 卖利益 | 学了能得到什么具体好处 | 直接驱动购买 |
| 卖痛点 | 不学会有什么损失 | 制造紧迫感 |

### 组合使用
- 种草视频：卖经历+卖认知（让用户觉得你有真东西）
- 转化直播：卖利益+卖好评（让用户觉得值）+ 卖痛点（让用户现在下单）
- 品牌视频：卖情怀+卖经历（让人记住你是谁）

## 五、定价策略

### 核心理念
**用户为「价值感」付费，不是为「价值」付费。感觉有了才有价值。**

### 定价三原则
1. **锚定定价**：先让用户知道「市场上的价格」，再给他「你的价格」
2. **阶梯定价**：低价体验课 → 中价训练营 → 高价私教
3. **价值感知**：不是卖课程节数，是卖「学完能多赚多少钱」

### 低价课策略
- 9.9-39 元体验课：不是为了赚钱，是为了建立信任
- 核心要求：**物超所值**——虽然便宜但内容要足够干
- 目标：体验课用户 → 正价课用户（复购转化）

### 高价课策略
- 卖的不是「知识」——知识可以免费获取
- 卖的是「确定性」——跟着我做，你也能拿到结果
- 卖的是「效率」——你自己摸索要 2 年，跟我学只要 2 个月

## 六、收入公式拆解

```
月收入 = 粉丝数 × 精准用户比例 × 直播间曝光率 × 点击率 × 转化率 × 客单价 × 复购率
```

### 优化优先级
1. **精准用户比例**（定位问题）→ 先确保粉丝是对的人
2. **转化率**（销售能力）→ 话术、信任、产品匹配
3. **客单价**（定价能力）→ 产品设计、价值包装
4. **复购率**（交付质量）→ 用户体验、长期关系
5. **粉丝数**（增长问题）→ 最后才考虑

### 关键结论
**收入不取决于粉丝量。10 万泛粉不如 1000 个精准粉。**

## 七、跨平台的变现思维

### 公私域共振
- **公域**（抖音/小红书/B站）：获取流量、建立信任、引流
- **私域**（微信/社群）：深度运营、高客单价转化、长期复购
- 在公域做成交，平台会给额外流量奖励——这是纯私域成交没有的好处

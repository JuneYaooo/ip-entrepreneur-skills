# 数据驱动体系

> 核心认知：数据是你的眼睛。没有数据就是盲人创业。在非线性增长时代，懂数据的人学习效率是百倍。

## 一、增长闭环

```
行动（做了什么）
  ↓
数据（产生了什么结果）
  ↓
认知（发现了什么问题）
  ↓
改进（调整了什么策略）
  ↓  循环
再行动
```

### 关键认知
- 「人在做，数据在看」——AI 引擎比你更了解你的内容表现
- 抖音的 AI 基于万人行为统计做判断，你的直觉会被数据打脸
- 数据不是用来证明自己对的，是用来发现自己错的

## 二、三阶段增长模型

### MVP（最小可行产品）
- 目标：验证「有人需要我做的东西」
- 动作：最小成本做一个可交付的产品，找人试用
- 指标：有没有人愿意付费？反馈是什么？
- 时间：越快越好，不要追求完美

### PMF（产品市场匹配）
- 目标：找到产品与市场的最佳匹配点
- 动作：根据 MVP 反馈持续迭代，直到用户主动推荐
- 指标：复购率、转介绍率、用户留存
- 标志：产品自己会「卖自己」

### SU（规模化增长）
- 目标：在 PMF 确认后再扩张
- 动作：标准化流程、扩大流量、组建团队
- 指标：营收增长率、市场份额
- 注意：没有 PMF 就扩张 = 加速死亡

## 三、核心指标体系

### 短视频指标
| 指标 | 公式/含义 | 健康值 |
|------|----------|--------|
| 5秒完播率 | 前5秒看完 / 总播放 | 越高越好 |
| 整体完播率 | 完整看完 / 总播放 | ≥40% |
| 互动率 | (赞+评+转+收) / 播放 | ≥3% |
| 粉丝成本 | 该视频带来的新增粉丝数 | <1元/粉 |
| 转粉率 | 新增粉丝 / 播放量 | ≥0.5% |

### 直播间指标
| 指标 | 公式 | 含义 |
|------|------|------|
| 停留时长 | 平均观看时长 | 内容吸引力 |
| 互动率 | 评论+点赞 / 在线人数 | 参与度 |
| UV 价值 | 成交金额 / 观看人数 | 流量利用效率 |
| 转化率 | 下单人数 / 观看人数 | 变现能力 |

### 转化漏斗（直播）
```
曝光量（多少人看到了你的直播间）
  ↓ 点击率
进入直播间（多少人点进来了）
  ↓ 停留率
商品曝光（多少人看到了商品）
  ↓ 点击率
创建订单（多少人点了下单）
  ↓ 成交率
成交（最终付款人数）
```
每个环节的流失就是你的优化空间。

### 核心营收公式
```
成交 = 流量 × 转化率 × 客单价
```
- 流量不足 → 优化短视频引流+直播时长+发布频率
- 转化率低 → 优化话术+信任背书+产品匹配
- 客单价低 → 优化产品设计+价值包装

## 四、数据分析实战

### 阶段策略：抄作业 vs 做测试
- **0-70 分阶段**：「抄作业」——直接复制对标账号的选题、结构、发布时间
- **70-90 分阶段**：「做测试」——A/B 测试不同开头/不同选题/不同时长
- **90 分以上**：「做自己」——数据告诉你什么有效，形成自己的风格

### 推荐工具
- 飞瓜数据：分析对标账号、查看行业趋势
- 婵妈妈：直播间数据分析、竞品监控
- 抖音创作者后台：基础数据免费查看

### 每日数据检查清单
1. 昨天发的视频数据（完播率、互动率）
2. 粉丝增长来源（哪条视频带来的？）
3. 评论区高频词（用户关心什么？）
4. 对标账号今天发了什么？

## 五、数据敏感度训练

### 常识 vs 数据
- 常识：这个视频我觉得挺好的 → 数据：5 秒完播率 20%，开头太差
- 常识：今天流量不好 → 数据：发布时段错了，目标用户在线时间是晚上 8-10 点

### 一个核心习惯
**每次发完内容，先看数据，再看评论，最后看感觉。顺序不能反。**

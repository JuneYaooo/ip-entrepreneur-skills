# 爆款内容体系

> 核心公式：人设 × 干货 = 无穷大。只有干货是「工具人」，只有人设是「花瓶」。

## 一、爆款底层逻辑

### 平台算法核心三指标
| 指标 | 基准线 | 说明 |
|------|--------|------|
| 5 秒完播率 | 越高越好 | 决定是否进入下一级流量池 |
| 整体完播率 | ≥40% | 内容是否有持续吸引力 |
| 互动率 | ≥3% | 点赞+评论+收藏+转发 / 播放量 |

### 流量池进阶机制
- 初始流量池（200-500 播放）：算法测试你的内容
- 数据达标 → 进入更大流量池（1000-5000）
- 数据不达标 → 停止推荐
- 爆款的本质是：每个流量池的数据都达标

### 「先垂直，后出圈」策略
1. 早期深耕一个细分领域，积累精准死忠粉
2. 当垂直领域粉丝到一定量级后，逐步扩展话题范围
3. 不要一上来就做泛内容——算法不知道把你推给谁

## 二、黄金选题模型

### 四要素
**精准用户 × 价值提供 × 热点主题 × 转化目标**

### 选题来源——三维掘金选题表

| 维度 | 挖什么 | 怎么挖 |
|------|--------|--------|
| 自身经历 | 踩过的坑、成功的方法、改变的过程 | 时间线梳理：过去→现在→未来 |
| 行业痛点 | 用户高频问题、常见误区、信息差 | 看同行评论区、搜行业关键词 |
| 用户评论 | 粉丝问什么、质疑什么、想要什么 | 每天翻自己评论区+同行评论区 |

### 选题的五层价值阶梯（马斯洛改编版）
1. **生理/安全层**：赚钱技巧、省钱方法、避坑指南 → 流量最大但粉丝价值低
2. **归属/社交层**：行业认同、圈层归属、情感共鸣 → 粉丝粘性高
3. **尊重/成长层**：认知升级、能力提升、方法论 → 高客单价变现
4. **自我实现/价值观层**：人生选择、底层思维、世界观 → 铁杆粉丝+高溢价

层次越高，粉丝越少但变现越容易。新手从第 1-2 层切入，积累后再往上走。

### 选题的「具体化」法则
- ❌ 「如何做好直播」——太抽象
- ✅ 「我打了 300 场直播，总结出 3 个让直播间人数翻倍的方法」——场景+数字+方法

## 三、用户移情——你写的不是内容，是用户的心事

### 用户移情七维看板
| 维度 | 问自己 |
|------|--------|
| ①目标与角色 | 用户想成为什么样的人？在什么处境中？ |
| ②需要完成的任务 | 用户每天要做什么？什么让他们焦虑？ |
| ③所看 | 用户在看什么内容？关注哪些号？ |
| ④所言 | 用户在评论区说什么？跟朋友聊什么？ |
| ⑤所做 | 用户实际做了什么？买了什么？ |
| ⑥所听 | 身边人在跟用户说什么？老板/家人/同行 |
| ⑦所想所感 | 用户内心真正的恐惧和渴望是什么？ |

### 核心洞察
- 很多专家不会说「人话」——满口术语，用户听不懂
- 用户移情图练几十遍，你就能拥有选题的读心术
- 用户不是买你的产品，是买「现在的自己→理想的自己」的路径

## 四、故事结构——好故事就是唤醒常识

### 好故事的两大元素
**逆境 + 斗争**

### 经典叙事结构
```
现状（平凡日常）
  ↓
冲突（出现挑战/危机）
  ↓
斗争（努力解决的过程）
  ↓
转折（关键突破/认知改变）
  ↓
结果（改变后的状态）
  ↓
启示（给观众的方法论/价值观）
```

### 案例
- 刘媛媛：「农村女孩→北大→超级演说家冠军」——逆境+斗争=超强人设
- 苹果 Think Different 广告：不展示产品，展示价值观认同

### 细节描写的威力
- ❌ 「我当时很努力」
- ✅ 「那段时间我每天早上 5 点起床，对着镜子练口播 2 小时，练了 100 天」

## 五、内容结构模板

### 123 干货结构
```
黄金 3 秒（钩子）：制造悬念或痛点——"90% 的人做 IP 第一步就错了"
↓
要点 1：核心观点或方法
要点 2：展开说明或案例
要点 3：补充或进阶技巧
↓
行动呼吁：引导点赞/关注/评论区互动
```

### 「黄金 3 秒」钩子类型
1. **反常识**：你以为的 X，其实是 Y
2. **数字冲击**：87% 的人都忽略的一个细节
3. **痛点直击**：做 IP 三个月没起色，问题在哪
4. **身份代入**：如果你也在做短视频……
5. **结果前置**：我用这个方法，一个月涨了 5 万粉

### 高转化销售视频结构（广告三步法）
1. **撩拨**（前 3-5 秒）：精准定位痛点，唤起需求——"你有没有遇到过 X 问题？"
2. **说服**（主体）：个人经历+方法论+案例对比，建立信任
3. **关单**（结尾）：明确行动指令+紧迫感——"限时/限量/现在"

## 六、拍摄与剪辑最低门槛

### 设备
- 不需要专业相机——各品牌旗舰手机足够
- 最关键的是**灯光**，不是相机
- 支架+环形灯 ≈ 200 元解决

### 拍摄参数
- 1080P 分辨率 + 60fps
- 关闭 HDR
- 人脸位于画面上半部分中轴线
- 人与背景保持距离（增加纵深感）

### 剪辑
- 快提词 APP 解决忘词问题
- 剪映足够处理所有需求
- 一个人就能完成所有操作

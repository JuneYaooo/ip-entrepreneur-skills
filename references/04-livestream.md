# 直播运营体系

> 直播是 IP 变现的放大器。粉丝量不重要，直播好才是真好。

## 一、直播间核心循环模型

### 吐故纳新循环
```
干货输出（拉人气：在线人数上升，没收入）
  ↓ 人气最高点
销售变现（做转化：收入上升，人气下降）
  ↓ 降到一定水平
干货输出（再拉人气）
  ↓
……循环往复
```

### 循环节奏
- **激进型**（带货直播间）：5-10 分钟一轮
- **保守型**（知识付费）：30-60 分钟一轮
- 核心规则：不要在人气最低的时候做转化，不要在转化做一半的时候突然切干货

### 游泳池模型
直播间在线人数 = 进场人数 - 离场人数
- 进场靠：流量入口（关注 Tab、短视频引流、推荐流）
- 离场靠：内容没吸引力、讲太久不互动、画面/声音质量差

## 二、开播前准备清单

### 必做项
- [ ] 选定主题（今天讲什么，卖什么）
- [ ] 写出逐字稿（至少：自我介绍、产品介绍、开场话术）
- [ ] 设置定时开播提醒
- [ ] 准备好实时数据看板（手机或电脑）
- [ ] 准备 2-3 个福袋/福利
- [ ] 测试画面、声音、灯光
- [ ] 准备 5 个互动问题（没人说话时自己抛）

### 逐字稿模板
```
【开场 2 分钟】
大家好我是 XX，今天直播我们聊一个话题：XXX
先跟大家说个事——（制造悬念）
觉得有用的打个「有用」

【干货输出 10-20 分钟】
第一点……
第二点……
第三点……
中间穿插："听懂的打「懂了」"

【转化 5 分钟】
刚才讲了 XXX，如果想深入系统学习……
（引导下单/加粉丝团/预约）

【循环回到干货】
```

## 三、人货场统一原则

### 人（主播人设）
- 你是什么人设，直播风格就要匹配
- 知识博主 → 专业但不枯燥
- 带货主播 → 亲切但专业
- 创业导师 → 实操接地气

### 货（商品/内容）
- 卖的东西要和你的定位一致
- 知识 IP 卖课，不要突然卖零食

### 场（直播场景）
- 直播间装修不必高级——信息清晰最重要
- 知识主播：书架背景 + 清晰的你
- 带货主播：产品展示清晰 + 价格信息醒目
- 背景干净＞花里胡哨

## 四、连麦高阶技巧

### 连麦的本质
**不是私聊——是舞台表演。你的目的是留住围观的人，不是只服务连麦的人。**

### 四步法
1. **破冰**（30秒）：自我介绍、拉近距离
2. **引导背书**（1分钟）：引导对方说出背景，增加可信度
3. **解决问题**（3-5分钟）：用对方的问题展示你的方法论
4. **泛化总结**（1分钟）：把这个个案变成通用方法——「不只是他，很多人都有这个问题，解决方法就是……」

### 连麦禁忌
- 不要只跟连麦的人聊天，忘了直播间其他人
- 不要被带偏话题
- 遇到杠精：借冲突做人设——「感谢你的质疑，我来告诉你我的逻辑」

## 五、关键转化技巧

### 价格锚点
- 先说原价/市场价，再说你的价格
- 「市面上类似的课程要 5000，我今天直播间只要……」

### 稀缺感营造
- 限量：「还剩 10 个名额」
- 限时：「这个价格只在直播间」

### 秒杀话术框架
```
【造势】：接下来这个东西，很多人等了很久
【产品介绍】：30 秒说清楚是什么
【原价铺垫】：平时卖 XXX
【直播价】：今天直播间 XXX
【紧迫感】：只有 X 个/X 分钟
【行动指令】：要的直接扣 1
```

### 沉默期怎么办
- 不是在直播间没人说话，是你要主动抛互动
- 「觉得有用的打个 1」
- 「是你你会怎么选？A 还是 B？打在公屏上」
- 5 分钟一个互动锚点

## 六、直播引流组合拳

### 时间轴策略
```
开播前 48 小时：发预告视频（剧透今天直播的干货）
开播前 0.5-1 小时：发预热视频（提醒直播马上开始）
直播中每 30 分钟：发一条切片（截取直播精彩片段引流）
直播后 2 小时：发复盘/精华剪辑
```

### 引流视频公式
```
"刚刚在直播间讲了一个方法/数据/八卦……
具体怎么回事？
点进头像，正在直播中"
```

## 七、直播间违规红线

### 高压线领域
- 宝妈创业（平台敏感）
- 玄学命理
- 医美健康断言
- 金融理财承诺收益

### 禁用词
- 不能说「微信」「QQ」「手机号」（用「小绿书」「地球号」「V」替代）
- 不能说「最」「第一」「绝对」
- 不能说「原价」然后直播价（除非你确实标注过原价）

### 避规策略
- 不挂车不要提商品
- 涉及平台规则的内容提前了解最新政策
- 录屏存证——防止被恶意举报

## 八、直播后复盘

### 复盘四步
1. **看曲线**：直播数据曲线——哪个时间点人流波动最大？
2. **对逐字稿**：实际讲了什么 vs 计划讲什么，偏差在哪
3. **看转化点**：哪个时间点成交最多？回看那个点你说了什么
4. **改进清单**：下次改 3 个点

### 核心原则
**录屏。每次直播都录屏。** 不录屏复盘等于白播。

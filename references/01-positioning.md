# IP 定位体系

> 核心原则：做 IP 就是做公司，一开始就要考虑盈利。IP 不是流量和曝光，而是商业模式。

## 一、三圈定位模型

找到三圈交集：**我擅长的 × 我喜欢的 × 能赚钱的**

### 擅长（能力圈）
- 列出你的专业技能、行业经验、特殊背景
- 不一定是顶尖水平——你只需要比目标用户懂就行
- 常见误区：过于谦虚，忽视「普通人看来很厉害」的能力

### 喜欢（动力圈）
- 什么话题你愿意一直讲、一直研究？
- 做 IP 是长期的事，不喜欢的内容撑不过死亡区

### 赚钱（商业圈）
- 这个领域有没有付费意愿？
- 客单价空间多大？
- 变现路径是什么（课程/咨询/带货/加盟）？

### 三圈不交叠时的策略
- 擅长+喜欢≠赚钱 → 先找 PMF，找到付费场景再重投入
- 擅长+赚钱≠喜欢 → 先用这个赚钱，再慢慢过渡到喜欢的
- 喜欢+赚钱≠擅长 → 快速学习，边学边做边输出

## 二、人设层次

### 人设不是编造——是挖掘和放大
人设 = 真实背景 + 独特观点 + 个人经历 + 性格特质

### 人设构建四步法
1. **背景梳理**：学历、工作经历、成就、失败经历
2. **观点提炼**：你对行业有什么独到看法？有什么反常识认知？
3. **经历选择**：选 3-5 个最能代表你的故事（逆境故事尤其有力）
4. **性格放大**：真实性格的一个侧面，放大到可辨识的程度

### 人设稀缺性 = 竞争壁垒
- 你的背景组合越独特，越难被复制
- 「北大博士 + 抖音创业」就比「抖音创业导师」稀缺得多
- 把你的「不务正业」变成你的「跨界优势」

## 三、对标账号分析

### 找 5 个同赛道对标账号
- 头部（>100 万粉）：看天花板，看变现模式
- 腰部（10-100 万粉）：看内容策略，看人设差异
- 起步期（1-10 万粉）：看起号路径，看早期内容

### 拆解维度
| 维度 | 要分析什么 |
|------|-----------|
| 人设 | 背景、观点、性格标签 |
| 内容 | 选题方向、内容形式、发布频率、爆款规律 |
| 直播 | 频次、时长、话术、转化方式 |
| 变现 | 卖什么、怎么卖、客单价 |
| 粉丝 | 评论区高频词、粉丝画像 |

### 防守点 vs 进攻点
- **防守（你不能被抄的）**：独特的背景和经历——这是你的护城河
- **进攻（你去抄别人的）**：内容结构、选题方法、运营技巧——这些可以学

## 四、账号装修——八维自检表

| 维度 | 标准 | 常见错误 |
|------|------|---------|
| 昵称 | 行业Title+称谓+名字，一眼知道你是干什么的 | 英文名、无意义数字、太长 |
| 头像 | 真人、清晰、有亲和力、与人设匹配 | 风景照、卡通、远景看不清脸 |
| 头图 | 四大功能：信任背书/价值主张/行动引导/品牌强化 | 空白、纯色、无信息量 |
| 简介 | 回答六个问题：你是谁/干什么的/为什么信你/提供什么/怎么联系/关注理由 | 写作文、无钩子 |
| 合集 | 按主题分类，降低新用户决策成本 | 无合集或合集太多 |
| 置顶视频 | 最能代表你的 3 条：一条爆款+一条自我介绍+一条转化 | 置顶无关内容 |
| 封面 | 统一风格、统一字体、统一色调 | 每期不一样 |
| 主页色调 | 视觉一致性，让人记住 | 杂乱无章 |

### 简介万能模板
```
①我是谁（一句话身份标签）
②我的背景/成就（信任背书）
③我提供什么价值（解决什么问题）
④为什么关注我（关注理由）
⑤联系方式/行动引导
```

## 五、IP 定位常见陷阱

1. **定位太宽**：「做创业内容」——太泛，没有记忆点 → 窄化到「北大博士教你用抖音创业」
2. **定位太窄**：「教人用 Excel 的 VLOOKUP 函数」——太小众，无法持续 → 扩展到「数据分析驱动个人成长」
3. **定位脱离变现**：「做搞笑段子」——粉丝多但变不了现 → 重新定位到可付费的领域
4. **人设造假**：编造经历迟早翻车 → 挖掘真实经历，不够就继续积累
5. **盲目模仿**：完全复制头部账号 → 防守点学不了，进攻点可以学
